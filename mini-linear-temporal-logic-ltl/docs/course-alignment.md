# LTL Course Alignment — Nine-School Curriculum Mapping

## MIT
- **6.841 Advanced Complexity Theory**: LTL satisfiability PSPACE-completeness (Sistla & Clarke 1985), automata-theoretic verification
- **6.820 Fundamentals of Program Analysis**: LTL model checking, SPIN model checker
- Covered: LTL syntax/semantics, model checking algorithm, satisfiability via tableau

## Stanford
- **CS254 Computational Complexity**: PSPACE-completeness of LTL problems
- **CS256 Formal Methods**: LTL specifications, model checking with NuSMV
- Covered: PSPACE results, model checking, pattern library

## Berkeley
- **CS278 Computational Complexity**: LTL in the polynomial hierarchy
- **CS294 Program Synthesis**: LTL specifications for synthesis (Pnueli & Rosner)
- Covered: Complexity results, GR(1) preparation via fairness patterns

## CMU
- **15-855 Graduate Complexity**: Automata-theoretic approach, ω-regular languages
- **15-414 Bug Catching**: LTL model checking in practice
- Covered: Tableau construction, counterexample extraction

## Princeton
- **COS 522 Computational Complexity**: LTL, CTL, and model checking complexity
- **COS 551 Advanced Complexity**: Synthesis and games
- Covered: Model checking, satisfiability, synthesis preparation

## Caltech
- **CS 151 Complexity Theory**: Temporal logic and verification complexity
- Covered: PSPACE-completeness results

## Cambridge
- **Part II Complexity Theory**: LTL and model checking
- **Part III Advanced Topics**: Automata on infinite objects
- Covered: LTL semantics, ω-words, tableau method

## Oxford
- **Computational Complexity**: LTL in verification
- **Advanced Complexity Theory**: Temporal logic decision procedures
- Covered: Tableau decision procedure, equivalence checking

## ETH
- **263-4650 Advanced Complexity**: LTL model checking, automata theory
- **252-0400 Logic & Computation**: Temporal logics in computer science
- Covered: LTL syntax/semantics, NNF, Fischer-Ladner closure, safety/liveness

## Key Textbooks
- Baier & Katoen 2008 — Principles of Model Checking (Ch. 5) — primary reference
- Clarke, Grumberg, Peled 1999 — Model Checking (Ch. 2, 9)
- Manna & Pnueli 1992 — The Temporal Logic of Reactive and Concurrent Systems
- Holzmann 2004 — The SPIN Model Checker
- Gradel, Thomas, Wilke 2002 — Automata, Logics, and Infinite Games
